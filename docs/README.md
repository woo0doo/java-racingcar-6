## **구현할 기능 목록**

---

### 입력 관련 기능

- 경주할 자동차 이름 입력 기능
- 시도할 회수 입력 기능

### 출력 관련 기능

- 게임 시작 문구(자동차 이름 입력 유도, 시도 회수 입력 유도)
- 각 차수별 실행 결과 출력
- 우승자 안내 문구

### 게임 관련 기능

- 차가 앞으로 나가는 기능
- 랜덤 숫자 생성 기능
- 랜덤 숫자가 4 이상일 경우 차 전진 기능
- 제일 앞서 나간 자동차 우승 판별 기능

### 검증 관련 기능

- 입력 기능 검증
  - 시도할 회수 숫자인지 검증
  - 시도할 회수 양수인지 검증
